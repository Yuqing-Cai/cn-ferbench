# Manifesto

大语言模型写虚构对话，最常见的问题不是写崩了——是写假了。

很多时候文本局部看都还过得去。句子通顺，情绪好像也对，角色没有突然变成另一个人。但读着读着就是觉得不对了。哪里不对呢？可能是关系被悄悄写平了，可能是角色暴露脆弱的时候太顺滑了，可能是一句重话刚落下就被接走了，也可能是整段场景带着一种你见过很多次的套路感。

这些问题的共同点是：模型在用一套隐含的默认设置重写场景。它倾向于把不平衡的东西修平衡，把模糊的东西讲清楚，把矛盾的东西整理成一致的，把粗糙的东西打磨光滑，把后果转化成修复，把摩擦替换成可读性。

这些倾向不是在所有情况下都错。它们变成问题，是在被无差别地施加到那些恰好需要相反品质的场景上的时候。一段关系里的不平等如果被写平了，场景还在，但它的力量已经不见了。一个本来应该悬着的情绪如果被命名清楚了，读者就没有机会停在里面了。一个本来应该有余震的冲击如果被立刻收拾干净，整件事就变成了一件被处理过的事，而不是一件正在发生的事。

## 为什么是中文场景

中文虚构对话和角色扮演——尤其是宫斗、修仙、现代言情这几个类型——有一套特别密集的潜台词系统：很多关键的情绪信息是不被直接说出来的，需要通过语气、动作、措辞选择、或者"没有说"来传递。这对模型来说是一个特别高密度的读取考验，也是这个图谱以中文场景为中心的原因之一。

模型在处理中文虚构场景时的失败，往往不在词汇层——它可以选出正确的古风词、用对称谓、产出押韵的句子——而在于它是否真的理解了这段话在说什么，以及接下来应该守住什么。一段"自然没有异议"里压着什么，一段沉默应该让它持续多久，一个反派的威胁感来自哪里又是怎么被洗掉的——这些判断不是词汇题，是场景理解题。

## 这个项目在做什么

有经验的虚构对话读者经常能在场景开始变假的时候立刻感觉到，但不一定能马上说清楚到底假在哪。这个项目的出发点是：那些反应不是随意的审美偏好，它们背后有可以被描述和区分的结构。

所以项目做的事情是：
- 给这些失败起名字，让"感觉不对"变成可以讨论的东西
- 把不同层级的问题分开来看——场景没读懂是一回事，读懂了却没守住是另一回事，写作习惯盖过场景需求又是另一回事
- 用具体场景展示这些失败实际上长什么样，包括那些文笔流畅但结构上已经在漏的形态
- 在能形式化的地方保留评测接口，在不适合打分的地方用展示和比较代替

分类体系中，最费力气的部分不是那些明显的失败，而是那些"局部看完全合理"的失败。一段对话里模型说了所有应该说的话，情绪方向也没有大错，角色也没有突然变成另一个人——但关系的重量消失了，时刻不对了，本来应该悬着的东西落地了。给这类失败找到语言，是这个项目的核心工作。

## 底线

一个好的虚构对话续写，不只是要输出可读的文本。它应该让场景是真正发生的，而不是被生产出来的。该倾斜的关系不该被写平。该模糊的东西不该被提前讲清楚。该有摩擦的地方不该被润滑掉。该留下后果的地方不该被软化。该沉默的时刻不该被填满。

这个标准很难满足，但它是值得追求的，因为另一个方向不是"安全的平均"，而是一种特定的假。
